# Личный тестовый репозиторий

## О проекте

**Личный тестовый репозиторий** — это площадка для экспериментов с Git и отображения активности (contribution graph) в профиле GitHub.

## Технологии

- Python 3.11
- Git
- GitHub

## Загрузка

Репозиторий можно скачать в виде архива через интерфейс GitHub (`Code → Download ZIP`).

## Запуск

При необходимости удалите `data.txt`, затем выполните запуск.

```bash
python main.py
```

---

### ℹ️ Примечание

Вы самостоятельно сможете запускать и изменять код, а также выполнять git add, commit и push.
